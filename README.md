# ⌨️ AutoTyper

A modern, dark-mode AutoTyper built with Python and CustomTkinter.  
Designed with a sleek Catppuccin-inspired UI, this tool automates typing with precision and style.

---

## ✨ Features

- 🎨 **Modern Dark UI** – Clean, minimal, Catppuccin-inspired design  
- ⏱ **Custom Delay** – Set typing delay (up to 45 seconds)  
- ⌨️ **Auto Typing Engine** – Simulates human-like typing  
- 🛑 **Global Stop Key** – Press `ESC` anytime to stop  
- ⚡ **Lightweight** – Fast and efficient  
- 📦 **Portable Build** – Compile into standalone executable or AppImage  


## 🚀 Getting Started

### 1. Clone Repository
```bash
git clone https://github.com/josva12/Autotyper.git
cd Autotyper
```
